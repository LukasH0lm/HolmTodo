# HolmTodo - Command Line Todo Application

A simple command-line todo application written in C.

## Features to implement:
- [ ] Add new todo items
- [ ] List all todos
- [ ] Mark todos as complete
- [ ] Delete todos
- [ ] Save/load from file

## Build Instructions:
```bash
gcc -o todo main.c
```

## Usage:
```bash
./todo add "Buy groceries"
./todo list
./todo complete 1
./todo delete 1
```
